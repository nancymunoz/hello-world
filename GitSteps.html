GitSteps

How to add a new file to a repo using a new branch and merging it back to main from within VS Code

  1. Open VS Code in the folder containing your repo
  2. Open the Command Palette by pressing Ctrl + Shift + P
  3. Type "Git: Create Branch" and select Git: Create Branch
  4. Name the new branch (e.g. new-file) and press Enter
  5. Add the new file in the new branch
  6. Open the Command Palette again and type "Git: Stage All" and select Git: Stage All
  7. Commit the changes by typing a commit message and pressing Enter
  8. Type "Git: Checkout to" and select Git: Checkout to...
  9. Select the main branch (e.g. main) and press Enter
  10. Type "Git: Merge Branch" and select Git: Merge Branch...
  11. Select the new branch (e.g. new-file) and press Enter
  12. Resolve any conflicts and commit the merge
